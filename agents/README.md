# Agents

## Purpose

This folder contains reusable AI agents used within the project.

Agents are structured prompts or workflows designed to perform specific tasks consistently.

---

## Rules

- Each agent must have a clear purpose
- Each agent must be reusable
- No temporary or experimental prompts
- Keep agents simple and focused

---

## Naming Convention

- `ticket-triage-agent.md`
- `data-cleaning-agent.md`
- `monitoring-agent.md`

---

## When to Create an Agent

Create an agent when:
- A task is repeated frequently
- The workflow benefits from standardisation
- Consistency is required across users
